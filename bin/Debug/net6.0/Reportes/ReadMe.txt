Instrucciones para Ejecutar la Prueba Automatizada:
Este programa realiza pruebas automatizadas en la plataforma O'Reilly Library Access utilizando Selenium WebDriver y ExtentReports para generar reportes y capturas de pantalla.

***REQUISITOS PREVIOS***

Asegúrate de tener instalado lo siguiente antes de ejecutar el programa:

Google Chrome
.NET Core SDK

***PASOS PARA EJECUTAR***

Clona este repositorio en tu máquina local.

Abre una terminal y navega a la carpeta del proyecto.

Compila el programa utilizando el comando dotnet.

Ejecuta el programa utilizando el comando dotnet.

Después de la ejecución, un reporte se generará en la carpeta "Reportes" dentro de la carpeta del proyecto. Abre el archivo "ExtentReport.html" en un navegador web para ver el reporte detallado de la prueba automatizada.

Las capturas de pantalla de cada acción realizada durante la prueba se guardarán en la carpeta "Screenshots" dentro de la carpeta del proyecto. Puedes revisar estas capturas para verificar las interacciones realizadas durante la prueba.

Una vez que hayas revisado el reporte y las capturas de pantalla, puedes cerrar el programa y la ventana del navegador.

***NOTAS***

Asegúrate de tener una conexión a Internet activa para que el programa pueda acceder a la plataforma O'Reilly Library Access.
Si experimentas algún error durante la ejecución de la prueba, revisa el reporte de error en el archivo "ExtentReport.html" y las capturas de pantalla en la carpeta "Screenshots" para identificar posibles problemas.
